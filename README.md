## 1.プロジェクトの概要

#### 1. プロジェクト名

NewsTwit

#### プロジェクトの目的

NewsPicksみたいなニュースにコメント付いている、サイトやアプリが好きなので、はてなブックマークのRSSでニュースのURLを取得して、TwitterのAPIとそのURLを紐つけて、ニュースに関するtweetを集めて、ニュースにコメントが付いているようなサイト・アプリを作りました。
